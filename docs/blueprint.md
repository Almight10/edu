# **App Name**: EduConnect

## Core Features:

- Google OAuth Login: Allow users to securely log in using their Google accounts.
- Dashboard: Personalized dashboard displaying relevant academic information such as courses, grades, and announcements.
- Course Management: Allow students to view enrolled courses, access course materials, and submit assignments.
- Gradebook: Enable students to view their grades and track academic performance.
- Announcements: Display important announcements and notifications from the institution or instructors.
- .env Configuration: Securely store Google OAuth API keys and other sensitive information in a .env file.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a trustworthy and academic feel.
- Background color: Light lavender (#E8EAF6), a very desaturated variant of the primary, to keep a professional, calming environment.
- Accent color: Muted violet (#7E57C2) to highlight important actions and notifications, creating good contrast and hierarchy.
- Body and headline font: 'Inter', a sans-serif with a clean, objective look.
- Use modern, clear icons for navigation and key actions, consistent with the geometric style.
- Clean, well-spaced layout with clear hierarchy to ensure easy navigation and readability.
- Subtle transitions and animations to provide feedback and enhance user experience.
